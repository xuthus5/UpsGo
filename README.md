## UpsGo

UpsGo是一个基于 Go+Vue 开发的又拍云 对象存储 服务管理应用。

### 构建

1. 重命名 config.yaml.example 为 config.yaml

2. 修改 config.yaml 内容

3. 运行

```bash
go run .
```
